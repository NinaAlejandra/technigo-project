  1. What is Node.js?

    Well it's a standalone server software based on the same JavaScript engine that Google uses in its Chrome browser. Node.js is an open source plattform, meaning that it's code designed to be publicly accessible for anyone to see, work with it and distribute. Node.js also lets you program javascript in its environment on the server. This means that you wont need multiple programming languages if you're working on the server-side and the client-side, you can just use Javascript!
  2. Companies that use Node.js
    Paypal, Uber, Linkedin and Netflix are some examples of companies that use Node.js.
